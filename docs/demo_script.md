# 演示脚本

> 本文档为评委提供建议的演示流程和提问示例，用于端到端体验项目核心能力。
>
> **演示环境说明：**
> - **公网 APK 演示**：直接打开 App，连接公网后端。
> - **本地部署演示**：先启动 Docker 后端，再用 Android Studio 运行 App。

---

## 1. 基础商品推荐

**用户提问：**

```
推荐几款跑鞋
```

**预期效果：**

- 助手返回简短推荐文本
- 展示 1-3 张商品卡片（跑步鞋）
- 每张卡片有真实的推荐理由（如 `匹配「轻量、缓震」需求，适合跑步训练。`）

> **公网 APK**：直接输入提问。
> **本地部署**：确认后端已启动，Emulator 中使用 `http://10.0.2.2:8080`。

---

## 2. 多轮上下文

**第 2 轮提问：**

```
要轻量的
```

**预期效果：**

- 仍然推荐跑鞋，不会跳到背包或电脑
- 继承上一轮 "跑鞋" 上下文

**第 3 轮提问：**

```
一千块以下的
```

**预期效果：**

- 继续继承跑鞋上下文，并加入预算约束
- 推荐价格不超过 1000 元的跑鞋

> **注意**：多轮上下文依赖 `session_id` 一致，公网后端和本地后端均支持。

---

## 3. 程序员电脑

**用户提问：**

```
推荐几款适合程序员的电脑
```

**预期效果：**

- 推荐笔记本电脑
- 推荐理由体现编程/多任务/大内存等需求
- 示例 reason：`匹配「编程、大内存」需求，适合编程开发。`

---

## 4. 页面上下文

**操作步骤：**

1. 进入某商品详情页（如一款护肤品）
2. 打开悬浮助手
3. 提问：

```
这个适合敏感肌吗？
```

**预期效果：**

- 围绕当前商品回答
- 不会推荐其他商品
- 回答基于当前商品的真实描述和 FAQ

---

## 5. 对话式加购

**用户提问：**

```
把第一款加入购物车
```

**预期效果：**

- 提示"已加入购物车：{商品名}，¥{price}，数量 ×1"
- 购物车角标自动刷新

---

## 6. 购物车问答

**用户提问：**

```
当前购物车多少钱了？
```

**预期效果：**

- 读取真实购物车金额
- 返回如"当前购物车共有 X 件商品，合计 ¥XXX"

---

## 7. 凑单推荐

**用户提问：**

```
如果要凑1000块，有没有推荐商品？
```

**预期效果：**

- 计算当前购物车与 1000 元的差额
- 推荐价格接近差额的商品
- 推荐理由包含"适合凑单"

---

## 8. 语音输入

**操作步骤：**

1. 长按悬浮助手的语音按钮
2. 说出：

```
推荐几款跑鞋
```

**预期效果：**

- 语音识别后自动发送
- 走正常 RAG 推荐流程
- 返回文本 + 商品卡片

> **注意**：某些 Android Emulator 不支持 ASR，建议用真机测试语音输入。

---

## 9. TTS 朗读

**操作步骤：**

1. 发送任意推荐请求
2. 等待 SSE 流结束（done 事件）
3. 助手自动朗读回答文本

**预期效果：**

- 听到 AI 语音朗读推荐内容
- 点击静音按钮可停止播放

> **公网 APK**：确保 `TTS_PUBLIC_BASE_URL` 为公网地址。
> **本地部署**：确保 `TTS_PUBLIC_BASE_URL` 为 `http://10.0.2.2:8080`。

---

## 10. 异常降级

**说明：**

- 如果 TTS 不可用（如 edge-tts 未安装或失败），文字回答和商品卡片仍可正常展示
- 如果 LLM API Key 未配置，可切换为 Mock LLM 模式体验基础流程
- 如果网络不稳定，SSE 会返回 error 事件，前端会展示错误提示

## 演示顺序建议

| 顺序 | 场景 | 目的 | 环境 |
|------|------|------|------|
| 1 | 基础商品推荐 | 展示核心 RAG 能力 | 公网/本地均可 |
| 2 | 多轮上下文 | 展示上下文继承 | 公网/本地均可 |
| 3 | 页面上下文 | 展示 PageContext-aware | 公网/本地均可 |
| 4 | 对话式加购 | 展示购物车集成 | 公网/本地均可 |
| 5 | 购物车问答 | 展示语义理解 | 公网/本地均可 |
| 6 | 凑单推荐 | 展示购物车增强 | 公网/本地均可 |
| 7 | 语音输入 | 展示多模态交互 | 建议真机 |
| 8 | TTS 朗读 | 展示语音输出 | 公网/本地均可 |
